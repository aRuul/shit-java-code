站在人文与AI的十字路口，当然选择拉屎💩。
展示 AI 时代的反叛精神 —— 专业生成、劣化、审查最糟糕的 Java 代码。

## 快速开始

在 VS Code Copilot Chat 中输入 `/` 然后选择 `shit-java-code`，或直接使用以下命令：

### 生成屎山代码
```

```

### 劣化现有代码
```

[粘贴你的优雅代码]
```

### 审查屎山程度
```
@workspace /shit-java-code 审查这段代码的屎山程度：

[粘贴待审查的代码]
```

## 技能特性

✅ **生成功能**：根据需求从零创建屎山代码  
✅ **劣化功能**：将优雅代码改写为屎山版本  
✅ **审查功能**：评估代码的屎山程度并给出详细报告  
✅ **教学价值**：每个反模式都有说明和示例  
✅ **丰富模式**：包含 50+ 种反模式组合  

## 包含的反模式类型

- 🏷️ **命名灾难**：单字母、拼音、数字后缀
- 📏 **方法暴政**：超长方法、深层嵌套、多职责
- 🏛️ **类混乱**：上帝类、贫血模型、过度继承
- 📝 **注释灾难**：无注释、废话注释、注释掉的代码
- ⚠️ **异常混沌**：吞噬异常、异常作为控制流
- 🔮 **魔法世界**：魔法数字、魔法字符串、硬编码
- 🧵 **并发灾难**：共享可变状态、线程不安全
- 🗄️ **数据库地狱**：SQL 注入、N+1 查询、忘记关闭连接
- 🏗️ **设计原则违反**：违反 SOLID、迪米特法则等
- 💀 **其他恶习**：全局变量、单例滥用、循环依赖
